# 临时要求

> 状态：**待开发**
> 此文件用于记录每月排班中的一次性或临时性特殊要求。
> 正式实现前，由开发者手动处理或在求解器外部补丁。

---

## 格式说明

每条临时要求按以下格式记录：

```
### [月份] 要求描述
- 类型：假期封锁 / 全体班主任 / 单人豁免 / 其他
- 影响范围：哪几天 / 哪些老师
- 处理方式：待开发 / 手动调整 / 已实现
```

---

## 示例（待实现）

### 5月 五一假期
- 类型：假期封锁
- 影响范围：劳动节放假期间（具体日期待确认）不安排任何值班
- 处理方式：待开发

### 6月 某日全体班主任到场
- 类型：全体班主任强制出勤
- 影响范围：指定日期，所有班主任必须值班
- 处理方式：待开发
